图片

单张图片

方式1：使用 img 标签，插入属性 data-fancybox。
方式2：使用Markdown语法引用图片，在全局引用Fancybox时 额外用js给所有图片插入属性 data-fancybox。

<a
  data-fancybox
  data-src="https://lipsum.app/id/1/1600x1200"
  data-caption="Hello world"
>
  <img src="https://lipsum.app/id/1/200x150" width="200" height="150" alt="" />
</a>

JSFiddle Example

组合图片

组合图片：使用a标签包裹img标签，并在a标签中插入属性 data-fancybox=”gallery” ，“gallery”为自定义的分组识别名

第一步：在Html中将图片分好组

<!-- First gallery -->
<a href="image-a1.jpeg" data-fancybox="gallery-a" data-caption="Gallery A #1">
  <img src="thumbnail-a1.jpeg" />
</a>

<a href="image-a2.jpeg" data-fancybox="gallery-a" data-caption="Gallery A #2">
  <img src="thumbnail-a2.jpeg" />
</a>

<!-- Second gallery -->
<a href="image-b1.jpeg" data-fancybox="gallery-b" data-caption="Gallery B #1">
  <img src="thumbnail-b1.jpeg" />
</a>

<a href="image-b2.jpeg" data-fancybox="gallery-b" data-caption="Gallery B #2">
  <img src="thumbnail-b2.jpeg" />
</a>

第二步：在JS中将图片的分组与Fancybox绑定一块

Fancybox.bind("[data-fancybox]", {
  // Custom options for all galleries
});

Fancybox.bind('[data-fancybox="gallery-a"]', {
  // Custom options for the first gallery
});

Fancybox.bind('[data-fancybox="gallery-b"]', {
  // Custom options for the second gallery
});

颜色

字体

字体家族 (Font-Family)

字体家族的选择首要考虑的是要适合屏幕的显示，并且至少要有轻、正常、重这三种字重可以选择。

有时为了突出或区分一些不同的内容，需要用到不一样的字体。

字重 (Font-Weight)

字体通过使用不同的字重，可以让内容表现出层级感。从轻到重一共设置5个层级，对于内容的区分有很大的什么用。

300 400 500 600
